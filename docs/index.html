MineSweepears island is a minesweeper inspired game with some twists

  • The island is composed of hex cells. 20% are trapped.
  • You score is based on sweep paths you make across the island. The number on a cell indicate both the number of adjacent trapped cells and how many point the cell is worth.
  • The longer the path the more points it brings. Cells can be used only once unless as when used as a starting point. You have 60 secondes

You have an an unconventional weapon at your disposal : your ears! When entering a tile mines in adjacent cells will play a random tune. The sound is spatialized, you can thus guess from which tiles it came from
